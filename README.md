# easycdebugger (ecd)

It's an easy C/C++ language debugger for Linux OS.

Requirements:<br>
> Python 3*<br>
> Linux *<br>
> gcc (You can install using sudo apt-get install gcc)

# How to Use

1 - Extract the ecd archive and allow it with chmod (chmod +x ecd);

2 - Command Usage: ./ecd <archive.cpp>
